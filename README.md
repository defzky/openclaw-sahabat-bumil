## ✨ What is Sahabat Bumil?

**Sahabat Bumil** is a pregnancy companion skill designed specifically for **Indonesian expectant mothers**. It provides:

- 🍎 **Indonesian nutrition guide** - Local foods, recipes, and dietary advice
- 🤢 **Morning sickness management** - Tips that work for Indonesian moms
- 🏥 **Hospital & BPJS info** - Navigate Indonesian healthcare system
- 💰 **Financial planning** - Cost estimates for Indonesian hospitals
- 🛒 **Preparation guides** - Hospital bag, baby registry, birth plan
- 💕 **Emotional support** - Especially for young moms (age 22-30)

---

## 🎯 Why Sahabat Bumil?

### **Made for Indonesian Moms**

Unlike generic pregnancy apps, Sahabat Bumil understands:

✅ **Local Foods** - Nasi, tempe, tahu, sayur bayam, etc.  
✅ **Cultural Context** - Traditions, pantang, confinement practices  
✅ **Healthcare System** - BPJS, Puskesmas, local hospitals  
✅ **Budget Reality** - Cost estimates for Indonesian hospitals  
✅ **Language** - Indonesian with English support  

### **Perfect for Young Moms**

Created with young mothers (age 22-30) in mind:

- Extra nutrition support (body still developing)
- Budget-friendly options
- Emotional support for first-time anxiety
- Career & motherhood balance tips

## 🚀 Quick Start

### **Installation**

```bash
# Via ClawHub (recommended)
openclaw skill install sahabat-bumil

# Or manual installation
git clone https://github.com/yourusername/sahabat-bumil.git ~/.openclaw/skills/sahabat-bumil
pip3 install -r ~/.openclaw/skills/sahabat-bumil/requirements.txt
```

```bash

/nutrition foods

# Learn about morning sickness
/nutrition morning-sickness

# Get hospital bag checklist

# Understand BPJS coverage

```

## 🤢 Special Feature: Morning Sickness

```

🤢 Morning Sickness Management
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🍽️ EATING TIPS:
• Eat small, frequent meals (5-6x/day)
• Keep crackers by bed
• Avoid empty stomach
• Choose bland foods

🌿 NATURAL REMEDIES:
• Ginger tea or ginger candies
• Peppermint tea
• Lemon water
• Vitamin B6 (consult doctor)

⏰ WHEN TO SEE DOCTOR:
• Can't keep food/fluids down
• Losing weight
• Signs of dehydration
```
